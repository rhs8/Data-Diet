How to read these numbers

Each food uses per-kg intensities from our dataset; weekly servings in the sidebar scale those values. Adding different metrics (GHG, land, energy, …) sums quantities with different units. Treat that sum as an exploratory index for comparison, not a single measurable outcome. For one interpretable measure, use GHG only (kg CO₂) below.
