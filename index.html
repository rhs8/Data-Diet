How to read these numbers

Each food uses per-kg intensities from our dataset; weekly servings in the sidebar scale those values. Adding different metrics (GHG, land, energy, …) sums quantities with different units. Treat that sum as an exploratory index for comparison, not a single measurable outcome. To focus on one measure (for example GHG in kg CO₂), turn off other impact types under Impact types in the sidebar. Same tap-to-toggle pattern as foods.
